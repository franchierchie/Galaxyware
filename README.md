
# Galaxyware

It's an ecommerce that sells PC-related products.




## Deployment

To deploy this project run

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open http://localhost:5173 with your browser to see the result.


## Technologies Used

 - React JS.
 - Redux Toolkit.
 - React Router.
 - Firebase - Firestore.
